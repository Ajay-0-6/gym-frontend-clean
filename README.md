# Gym Website 💪

Modern gym website built with React, Node.js, Express, and MongoDB.

## Features
- Responsive modern UI
- Membership join form
- Backend API
- MongoDB data storage
- Clean landing pages

## Tech Stack
- React + Vite
- Tailwind CSS
- Node.js
- Express.js
- MongoDB Atlas
